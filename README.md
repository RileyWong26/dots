## Setting up
#### Link zsh
```bash
# zshrc is normally in ~/
ln -s /dots/.zshrc ~/.zshrc
```

#### Link ghostty
```bash
# ghostty config is in ~/.config/
ln -s /dots/ghostty ~/.config/ghostty
```

#### Link neovim
```bash
# neovim config is in ~/.config/
ln -s /dots/nvim ~/.config/nvim
```
